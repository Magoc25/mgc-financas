⚙ Configuração — Supabase

Passo 1 — Criar tabela no Supabase

No painel do seu projeto Supabase, vá em SQL Editor e execute o script abaixo:

create table if not exists finance_data ( id integer primary key default 1, payload jsonb not null default '{}', updated_at timestamptz default now() ); grant select, insert, update, delete on finance_data to anon; grant select, insert, update, delete on finance_data to authenticated; grant select, insert, update, delete on finance_data to service_role; alter table finance_data enable row level security; create policy "public_access" on finance_data for all using (true) with check (true); insert into finance_data (id, payload) values (1, '{"rendas":[],"faturas":[],"fixas":[],"dividas":[],"meta":0}') on conflict (id) do nothing;

Cenários salvos

📚 Aprender

Renda fixa é como "emprestar" seu dinheiro (a um banco ou ao governo) e receber de volta com juros. É mais previsível que a Bolsa e ótima para reserva e metas. Abaixo, cenários prontos para começar e um glossário rápido.

🎯 Cenários prontos

📖 Glossário

CDI

Taxa de referência dos empréstimos entre bancos. A maioria dos investimentos pós-fixados rende um percentual dele (ex.: 100% do CDI). Anda colado na Selic.

Selic

Taxa básica de juros da economia, definida pelo Banco Central. É a base do Tesouro Selic e referência para o CDI.

IPCA

Índice oficial da inflação no Brasil. "IPCA+" significa render a inflação MAIS uma taxa fixa, protegendo o poder de compra.

Prefixado × Pós-fixado

Prefixado = taxa fixa travada na aplicação (você já sabe quanto vai receber). Pós-fixado = rende um % de um indexador que varia (ex.: 100% do CDI); o resultado depende de como o indexador se comportar.

IR regressivo

Imposto sobre o rendimento de aplicações tributadas. Cai com o tempo: 22,5% (até 180 dias), 20%, 17,5% e 15% (acima de 720 dias). Quanto mais tempo investido, menos imposto.

Poupança

Isenta de IR e com liquidez, mas costuma render menos (~0,5% ao mês + TR). Útil como referência de comparação.

CDB

Título emitido por bancos; rende % do CDI ou taxa prefixada. Tem IR (regressivo) e é protegido pelo FGC.

LCI / LCA

Títulos de crédito imobiliário/agro, ISENTOS de IR para pessoa física. Por isso uma LCI a 95% do CDI pode render mais que um CDB a 100%.

Tesouro Direto

Títulos do governo federal. Tesouro Selic: pós-fixado, liquidez diária, ideal para reserva. Tesouro IPCA+: protege da inflação, ideal para longo prazo. Ambos têm IR.

FGC

Fundo Garantidor de Créditos: garante até R$ 250 mil por CPF e por instituição em poupança, CDB e LCI/LCA, caso o banco quebre.

Liquidez

Facilidade de resgatar o dinheiro sem perder valor. Tesouro Selic e poupança têm alta liquidez — ideais para emergências.
