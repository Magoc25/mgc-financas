⚙ Configuração — Supabase

Passo 1 — Criar tabela no Supabase

No painel do seu projeto Supabase, vá em SQL Editor e execute o script abaixo:

create table if not exists finance_data ( id integer primary key default 1, payload jsonb not null default '{}', updated_at timestamptz default now() ); alter table finance_data enable row level security; create policy "public_access" on finance_data for all using (true) with check (true); insert into finance_data (id, payload) values (1, '{"rendas":[],"faturas":[],"fixas":[],"dividas":[],"meta":0}') on conflict (id) do nothing;
